# OpticAI 👁️
**High-Fidelity Perceptual Benchmarking & Visual Memory Engine**

OpticAI is a minimalist, high-stakes vision benchmark designed to quantify the limits of human color perception. By bridging the gap between brand strategy and technical execution, it challenges users to retain and recreate complex HSL spectrums under high-velocity constraints.

### 🚀 [Live Benchmark: https://opticalgame.vercel.app/]

### 🎯 Strategic Vision
Developed as a "Perceptual Audit" tool, OpticAI explores the intersection of **Computer Vision** and **Human Cognition**. It gamifies the principles of the Munsell Color System to quantify how accurately we retain visual equity (color) under psychological and temporal pressure.

### 🛠️ Tech Stack & Engineering
* **Engine:** React 18 (Functional Components & Hooks)
* **Styling:** Tailwind CSS + Custom Neomorphic "Old Money" CSS Architecture.
* **Algorithm:** 3D Euclidean Distance with a **1.8x Cubic Decay Scoring Engine** to elevate the skill ceiling.
* **UX Safety:** Integrated **Mobile-Touch Protection** and **Ghost-Timer Logic** for a zero-friction production experience.

### ✨ Key Features (Version 1.0)
* **Universal Glow UI:** High-intensity "radiant" neomorphic buttons to drive primary user actions.
* **Precision Benchmarking:** A definitive **50.00 Point Scale** with instructional "Memorise & Guess" headers for clinical clarity.
* **The Elite Floor:** Integrated logic that sets a **25-point minimum benchmark**, challenging users to reach professional-grade accuracy.
* **Viral Loop:** Dual-action sharing console (Direct Link + Social Score Card) designed for high-conversion social proof.
* **Visual Calibration:** Post-game split-view comparison cards to help users visually audit their perceptual errors.

### 🧬 Benchmark Stages
1. **Memorise the Colour:** 5.00 seconds of high-intensity color burning (retinal persistence).
2. **Guess the Colour:** 30.00 seconds of HSL manipulation via precision sliders.
3. **The Audit:** Real-time scoring against the original target with 0.01 decimal accuracy.
