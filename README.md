# OpticAI 👁️ 
**High-Fidelity Perceptual Benchmarking & Visual Memory Engine**

OpticAI is a minimalist, high-stakes web application designed to test the limits of human color perception. By bridging the gap between brand strategy and technical execution, it challenges users to retain and recreate complex HSL spectrums under high-velocity constraints.

### 🚀 [https://opticalgame.vercel.app/]

### 🎯 Strategic Vision
Developed as a "Perceptual Audit" tool, OpticAI explores the intersection of **Computer Vision** and **Human Cognition**. It gamifies the principles of the Munsell Color System to quantify how accurately we retain visual equity (color) under pressure.

### 🛠️ Tech Stack
* **Engine:** React 18 (Functional Components & Hooks)
* **Styling:** Tailwind CSS + Custom Neomorphic CSS Architecture
* **Algorithm:** 3D Euclidean Distance with 1.8x Cubic Decay Scoring
* **Deployment:** Vercel / GitHub Actions

### ✨ Key Features
* **Neomorphic UI:** A tactile, "Old Money" minimalist aesthetic.
* **Viral Loop:** Integrated "Share Vision" console with Wordle-style result copying.
* **Precision Analytics:** Split-view comparison cards for real-time visual calibration.
* **Adaptive Difficulty:** 5 distinct thematic rounds (Neon → Pastels → Muted Earth).
