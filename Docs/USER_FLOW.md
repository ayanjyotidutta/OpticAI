# 🗺️ User Journey Map: OpticAI
**Version:** 1.5 (Consistent Brand Build)

## 1. The Entry Point (The Hook)
* **Touchpoint:** Landing Dashboard.
* **Visual Strategy:** The **Neomorphic "Start" CTA**. By using a high-contrast white body with a circular action glyph, we create an immediate "Physical" pull. 
* **User Action:** Clicks "Start OpticAI" — the tactile `active:scale-95` transition provides instant haptic-style feedback.

## 2. The Core Loop (Perception Test)
* **Memorize (5s):** Total focus on the HSL target. The heartbeat-synced millisecond timer drives cognitive urgency.
* **Execute (30s):** User adjusts sliders. The **Glassmorphic Control Panel** ensures the user can see the changing background color while maintaining UI legibility.
* **Verification:** User clicks "Lock it In."

## 3. The Climax (Results & Analysis)
* **Visual Calibration:** The summary screen displays **Split-View Comparison Cards**. This is the "Learning Moment" where the user identifies their specific color bias (Hue vs. Saturation).
* **Status Assignment:** Custom algorithm assigns a Tier (e.g., "Divine Visionary") based on the 1.8x Cubic Decay score.

## 4. The Exit & Viral Loop (Social Equity)
* **Touchpoint:** The **Neomorphic "Share" CTA**.
* **Branding Consistency:** By mirroring the "Start" button’s architecture (Title Case + Circular Glyph), the UX feels "complete."
* **Action:** User clicks "Share OpticAI." The system bundles the Tier, Score, and Link into a "Wordle-style" text block for professional sharing.
* **Retention:** Adaptive CTA calculates a personalized goal: *"Try Beating [Score + 1] Score."*

### 5.🎨 Strategic UI Framework: "The Neomorphic Bookend"
We implemented a consistent button architecture across the **Start** and **End** states. 
1. **Symmetry:** Identical button shapes and iconography create a sense of professional "Finish."
2. **Tactile UX:** Both buttons use dual-source shadows to simulate physical hardware, elevating the tool from a "web game" to a "benchmark product."
